# Supply-chain-predictive-model

Este proyecto fue desarrollado como tesis de grado para optimizar la cadena de suministro del Centro de Acopio del Banco de Alimentos de Bogotá. El objetivo principal fue reducir la incertidumbre en la entrada de suministros mediante el desarrollo de un sistema predictivo capaz de estimar el volumen de donaciones.

La implementación de este modelo permite una mejor planificación en la recolección, almacenamiento y posterior distribución de alimentos, impactando directamente en la eficiencia operativa de la organización.

🛠️ Tecnologías Utilizadas

Lenguaje: Python 3.x

Librerías de Análisis: Pandas, NumPy

Visualización: Matplotlib, Seaborn, Power BI

Machine Learning: Scikit-learn (especifica si usaste Regresión Lineal, Random Forest, etc.)

Entorno: Jupyter Notebooks

📊 Metodología Aplicada

Extracción y Limpieza de Datos: Procesamiento de registros históricos de donaciones para eliminar ruidos y datos atípicos.

Análisis Exploratorio de Datos (EDA): Identificación de tendencias estacionales y variables críticas que afectan el volumen de donaciones.

Modelado Matemático: Aplicación de algoritmos de regresión/series de tiempo para la predicción de volúmenes.

Validación: Evaluación del modelo mediante métricas de error (MAE, RMSE) para asegurar la confiabilidad de las predicciones.

🚀 Impacto y Resultados

Optimización Logística: Proporcionó una herramienta base para la toma de decisiones en la asignación de rutas de recolección.

Reducción de Desperdicios: Al predecir el volumen entrante, se mejoró la gestión de inventarios y la rotación de productos perecederos.

Eficiencia Operativa: Recomendaciones técnicas basadas en datos para mejorar las prácticas de higiene y flujo de trabajo en el Centro de Acopio.
